# Design

## Metadata

- Product: Captrix
- Register: product
- Surface: browser-based creator caption studio

## Mood

Pre-dawn edit bay: crisp light surface, cobalt instrument glow, precise controls, vivid caption energy.

## Color

Use OKLCH tokens defined in `app/globals.css`.

## Typography

Use one system sans stack for speed and reliability. Product headings use fixed rem sizes, not viewport-fluid scaling.

## Layout

The primary composition is a two-column studio: large video preview on the left, dense controls on the right, metrics under preview, stacked on mobile.

## Components

Upload zone, caption overlay, style chips, position segmented control, local AI coach, export guard.

## Motion

Motion is limited to caption preview feedback and interaction transitions. All animations respect reduced-motion preferences.
